# Datadog Synthetics

Monitor API endpoints and browser workflows with automated testing.

## Modules

### [API Test](./api-test/README.md)
HTTP/HTTPS endpoint monitoring with:
- GET, POST, PUT, DELETE methods
- Custom headers and body
- Response time assertions
- Status code and body content validation
- Multi-location testing

### [Browser Test](./browser-test/README.md)
Browser-based user journey testing with:
- Multi-step workflows
- Click, type, and navigation actions
- Element presence assertions
- Screenshot capture on failure
- Multiple device types
