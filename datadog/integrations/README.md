# Datadog Integrations

Connect Datadog with cloud providers and third-party services.

## Modules

### [AWS Integration](./aws/README.md)
AWS CloudWatch metrics and logs with:
- IAM role-based authentication
- Namespace filtering
- Tag-based host filtering
- Region exclusion

### [Azure Integration](./azure/README.md)
Azure Monitor integration with:
- Service principal authentication
- Resource group filtering
- Tag-based host filtering

### [GCP Integration](./gcp/README.md)
GCP Stackdriver integration with:
- Service account authentication
- Project-level monitoring
- Host filtering
- Automute for maintenance

### [Slack Integration](./slack/README.md)
Slack notifications with:
- Multi-channel support
- Customizable message display
- Snapshot inclusion
- Tag display options

### [PagerDuty Integration](./pagerduty/README.md)
PagerDuty incident management with:
- Multi-service support
- API token authentication
- Automatic incident creation
