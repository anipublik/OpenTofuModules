# Datadog Dashboards

Visual dashboards for metrics, traces, and logs.

## Modules

### [Timeboard](./timeboard/README.md)
Time-synchronized dashboards with:
- Ordered widget layout
- Automatic time synchronization across widgets
- Template variables for filtering
- Query value, timeseries, and heatmap widgets

### [Screenboard](./screenboard/README.md)
Free-form dashboards with:
- Flexible widget positioning
- Custom layouts and sizing
- Note widgets for documentation
- Image and iframe widgets
