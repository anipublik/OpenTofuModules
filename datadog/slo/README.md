# Datadog SLO

Define and track Service Level Objectives with error budgets.

## Modules

### [Metric SLO](./metric/README.md)
Metric-based SLO tracking with:
- Custom numerator/denominator queries
- Multiple timeframes (7d, 30d, 90d)
- Warning and target thresholds
- Error budget tracking

### [Monitor SLO](./monitor/README.md)
Monitor-based SLO tracking with:
- Multiple monitor aggregation
- Uptime calculation
- Warning and target thresholds
- Error budget alerts
